## 01. コマンド

### global-status

起動中の仮想サーバを一覧で表示する。

参考：https://www.vagrantup.com/docs/cli/global-status

```bash
$ vagrant global-status
```

<br>

### halt

仮想サーバーを停止する。

参考：https://www.vagrantup.com/docs/cli/halt

```bash
$ vagrant halt
```

<br>

### ssh

仮想サーバーにSSH接続を行う。

参考：https://www.vagrantup.com/docs/cli/ssh

```bash
$ vagrant ssh
```

<br>

### up

仮想サーバーを起動する。

参考：https://www.vagrantup.com/docs/cli/up

```bash
$ vagrant up
```
